[INIT] 프로젝트 초기 세팅(첫 셋업, 폴더, 씬, 패키지)

[ADD] 기능/파일 추가(새 기능, 새 스크립트, 새 UI)  : 없던 기능이 생김

[REMOVE] 삭제(기능/파일 제거)

[RENAME] 이름 변경(클래스/파일/폴더명)

[REFACTOR] 리팩터(동작 동일, 구조만 개선) : 동작은 그대로, 코드 정리만 (함수 분리, 중복 제거)
